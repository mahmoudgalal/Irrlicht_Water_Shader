# Irrlicht_Water_Shader
 
 Legacy Directx9 Water Shader i developed for irrlicht3d game engine. 
 The shader is based on <b>ATI RenderMonkey Water-Shader(assembly)</b> & <b>NVIDIA Ocean Shader(HLSL)</b>.
 ### Tools Used:
 - VC++ 2019
 - Irrlicht3d v1.8.5
 - DirectX SDK
